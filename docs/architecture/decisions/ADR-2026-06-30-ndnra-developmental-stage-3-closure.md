# ADR: NDNRA Developmental Network Stage 3 Closure

Date: 30 June 2026
Status: Accepted implementation evidence after final gates
Scope: NDNRA v0.2 Stage 3 specialised concurrent regions and multiple needs

## Context

Stage 2 closed deterministic associative need-and-context recall over separately inspectable experiences. The next required boundary was Stage 3: prove that multiple needs and specialised regions can remain concurrently represented without reducing the system to one uniform network or one hidden scalar ranking.

## Decision

Accept Stage 3 as implemented and evidenced by:

- `src/seedmind/research/ndnra/developmental_regions.py`;
- `tests/unit/test_ndnra_developmental_regions.py`;
- `docs/architecture/NDNRA_Developmental_Stage_3_Evidence_2026-06-30.md`.

The accepted Stage 3 boundary is:

- deterministic and in memory only;
- based on region-local neuron pools and plasticity controls;
- explicit about multiple simultaneous need pulses;
- explicit about typed cross-region messages;
- able to preserve compatible task, resource, and clarity needs together;
- able to inhibit an incompatible destructive internal proposal through protected permission and preservation needs;
- able to let a dormant curiosity need re-emerge after a stop need resolves;
- explicit about uniform-network interference controls;
- explicit that no permanent global scalar may collapse all needs into one ranking;
- non-persistent;
- non-SQLite;
- zero-authority;
- disconnected from runtime adapters and action gateways.

## Consequences

- Stage 4 regional child-to-adult development, skill maturation, and ambition persistence is now the next bounded implementation target after the Stage 3 commit.
- Stage 3 does not prove maturity-dependent plasticity, skill promotion, persistent ambition, verifier maturity, homeostasis, hibernation, dream maintenance, restart identity, learned responsibility, typed external action proposals, shadow trial behavior, or any action gateway behavior.
- Regional cooperation is accepted only through inspectable typed messages and internal proposals, not through a hidden global quality scalar.
- Production curiosity remains the sole production action authority.
- The expanded developmental architecture marker remains 82%.
- No automatic push is authorised.
