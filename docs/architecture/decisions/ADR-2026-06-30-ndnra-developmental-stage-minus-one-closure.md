# ADR: NDNRA Developmental Network Stage -1 Closure

Date: 30 June 2026
Status: Accepted implementation evidence after final gates
Scope: NDNRA v0.2 Stage -1 developmental constitution contracts

## Context

The accepted v0.2 developmental-network plan requires an organising constitution before recurrent substrate work begins. Stage -1 had to define DESA, protected signal roles, ordinary and Executive Nursery curricula, desired-state ambition, capability-gap evidence, developmental skill bundles, Outcome Fidelity, bounded feedback iteration, protected authority, integrity protection, and candidate causal responsibility.

The implementation had to remain isolated under `src/seedmind/research/ndnra`, preserve closed v0.1 evidence, expose local and regional metacognitive summaries without central whole-network control, avoid SQLite cognition, avoid runtime integration, and grant no production action authority.

## Decision

Accept Stage -1 as implemented and evidenced by:

- `src/seedmind/research/ndnra/developmental_constitution.py`;
- `tests/unit/test_ndnra_developmental_constitution.py`;
- `docs/architecture/NDNRA_Developmental_Stage_Minus_One_Evidence_2026-06-30.md`.

The accepted Stage -1 contract boundary is:

- deterministic and in memory only;
- typed and publicly exported through `seedmind.research.ndnra`;
- non-authoritative;
- non-persistent;
- non-SQLite;
- separate from closed v0.1 proof stores;
- explicit that DESA contains no task solution, pretrained language model, imported task knowledge, runtime adapter, action gateway, or production action authority.

## Consequences

- Stage 0 is now the next bounded implementation target.
- Stage 0 must define v0.2 identity, lifecycle, deterministic trace, and schema-separation contracts without connecting runtime integration or persistence-store merging.
- Stage -1 evidence does not prove recurrent experiential memory, restart identity, DESA learning, associative recall, simultaneous needs, hibernation, dreaming, learned responsibility, safe proposals, or shadow trial behavior.
- Production curiosity remains the sole production action authority.
- The expanded developmental architecture marker remains 82%.
- No automatic push is authorised.
