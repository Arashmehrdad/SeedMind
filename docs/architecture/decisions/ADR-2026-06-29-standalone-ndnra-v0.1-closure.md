# ADR: Standalone NDNRA v0.1 Closure

Date: 29 June 2026
Status: Accepted
Scope: standalone NDNRA v0.1 closure only

## Context

Standalone NDNRA evidence has been completed across deterministic acceptance aggregation, isolated acceptance restart proof, normalized competing recruitment, locally derived representational saturation, long-horizon mixed-task interference, isolated long-horizon restart proof, and bounded imagination stage closure.

The remaining task is to lock the accepted closure boundary precisely so closure does not overclaim broader NDNRA research completion, SeedMind roadmap completion, integration approval, or unified runtime persistence.

## Decision

Standalone NDNRA v0.1 is accepted as closed only under this explicit scope:

- one active need pulse at a time;
- grounded exact context signatures;
- fixed bounded inspectable contextual-transfer weights;
- specialist interaction growth rather than unrestricted generic neuron creation;
- no semantic abstraction claim;
- no simultaneous multi-need arbitration claim;
- no production integration or production action authority;
- no scheduler, executor, autonomous recommendation, selection, promotion, or live Nursery authority;
- bounded imagination remains deterministic, in-memory, non-evidentiary, and non-authoritative.

Closure means the accepted standalone v0.1 research architecture has no remaining implementation blockers inside this declared claim boundary.

Closure does not mean:

- all future NDNRA research is complete;
- NDNRA has been compared against the original SeedMind plan;
- NDNRA is approved for integration;
- production action authority exists.

Persistence remains separated:

- standalone acceptance persistence and restart proof remain one isolated proof store;
- long-horizon interference persistence and restart proof remain a second isolated proof store;
- neither store is a unified cognitive runtime checkpoint;
- the main brain schema remains unchanged by this closure;
- bounded imagination is not persisted;
- missing, corrupt, or incompatible persistence input must return explicit non-proof fallback rather than partial recovery.

Zero-authority boundaries remain locked:

- no production action selection;
- no recommendation authority;
- no scheduler authority;
- no executor authority;
- no autonomous selection or promotion;
- no live Nursery authority;
- no SQLite cognition.

Deferred research remains post-v0.1:

- learned context-similarity weights;
- semantic abstraction;
- simultaneous multiple needs.

The expanded developmental architecture marker remains 82%. Closure consolidates accepted evidence but does not add new cognitive architecture capability.

## Consequences

- Standalone NDNRA v0.1 may now be treated as closed only within the declared research boundary.
- Integration-facing observer or shadow evidence remains boundary evidence only, not production approval.
- The next required work is comparison against the original SeedMind master plan, followed by an explicit retain, adapt, partially integrate, or do-not-integrate decision.
- No integration implementation is authorized before that later explicit decision.
