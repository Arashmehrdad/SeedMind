# ADR: NDNRA Developmental Network Stage 1A Closure

Date: 30 June 2026
Status: Accepted implementation evidence after final gates
Scope: NDNRA v0.2 Stage 1A DESA bootstrap and hierarchical metacognition

## Context

Stage 1 closed the recurrent experiential substrate: a fixed reusable neuron pool, distributed overlapping coalitions, signed recurrent connections, deterministic replay, and zero action authority. The next required boundary was Stage 1A: test whether the minimal DESA hierarchy can route, summarize, incubate temporary skills, calibrate verifiers, use event boundaries, and accept later correction without becoming a task solver or production authority path.

## Decision

Accept Stage 1A as implemented and evidenced by:

- `src/seedmind/research/ndnra/developmental_desa_bootstrap.py`;
- `tests/unit/test_ndnra_developmental_desa_bootstrap.py`;
- `docs/architecture/NDNRA_Developmental_Stage_1A_Evidence_2026-06-30.md`.

The accepted Stage 1A boundary is:

- deterministic and in memory only;
- layered on the Stage 1 substrate;
- limited to local monitors, regional captains, DESA council summaries, optional steward gates, temporary skill incubation, bounded feedback iteration, verifier calibration, event partitioning, Executive Auditor correction, and temporary ambition evidence;
- explicit about controls and ablations;
- non-persistent;
- non-SQLite;
- zero-authority;
- disconnected from runtime adapters and action gateways.

## Consequences

- Stage 2 associative need-and-context recall is now the next bounded implementation target after the Stage 1A commit.
- Stage 1A does not prove broad associative recall, simultaneous multiple needs, mature skill promotion, mature ambition lifecycle, hibernation, dreaming, restart identity, learned responsibility, typed action proposals, shadow trial behavior, or any action gateway behavior.
- DESA remains a metacognitive routing and calibration structure, not an all-knowing task solver or external authority owner.
- Temporary skill-bundle evidence remains developmental and non-authoritative.
- Pending or unavailable feedback remains unverified and does not mutate factual evidence.
- Production curiosity remains the sole production action authority.
- The expanded developmental architecture marker remains 82%.
- No automatic push is authorised.
